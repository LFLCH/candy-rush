## Prerequisite
A recent node version (20+).

Install the dependencies with :
```bash
npm install
```

## Commands
```bash
# build : compile .ts files into .js files
npm run build

# start : launch server
npm run start

# serve : build + start
npm run serve 

# dev : build and start on each file change
npm run dev
```
