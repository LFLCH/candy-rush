## Run

Simply with :
```bash
npm run start
```
And a click on the indicated ip adress in the console should open the browser and let you play.

## Test
The tests are done in the browser, thanks to [mocha](https://mochajs.org/) and [chai](https://www.chaijs.com/).

Do not forget to install the dependencies before :
```bash
npm install
```
Then,
```bash
npm run test
```
